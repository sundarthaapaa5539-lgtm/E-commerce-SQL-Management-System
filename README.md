# MegaStore E-commerce Database System 🚀
**Developed by:** thapaasundar

## 📌 Project Overview
This project involved designing and implementing a robust relational database for an e-commerce platform called **MegaStore**. The system is built to handle high-scale operations, including inventory management, customer tracking, and automated sales reporting.

## 🛠 Key Features & Technical Skills
- **Data Scaling:** Successfully managed and stored over **1,500+ products** and **850+ customer orders**.
- **Database Automation:** Developed **SQL Stored Procedures** to automate the generation of large testing datasets.
- **Business Intelligence:** Created advanced queries to track **Monthly Revenue**, **Top-Selling Products**, and **Low-Stock Alerts**.
- **Data Integrity:** Implemented foreign key constraints and data cleaning scripts.
